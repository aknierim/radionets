***************************************************
Deep-Learning Framework (`~radionets.dl_framework`)
***************************************************

.. currentmodule:: radionets.dl_framework

The used deep learning framework is based on `pytorch <https://pytorch.org/>`_
and `fastai <https://www.fast.ai/>`_. An introduction to Neural Networks and an
overview of the use of fastai to train deep learning models can be found in
`Practical Deep Learning for Coders, v3 <https://course.fast.ai/index.html>`_
and `fastbook <https://github.com/fastai/fastbook>`_.
